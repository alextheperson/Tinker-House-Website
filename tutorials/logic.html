Logic gates are the basis of all computing, you give them True(1) or False(0) and they give you True of False back. These are the basic logic gates:

NOT Gate (¬p)

Input Output
True False
False True

AND Gate (p∧q)

Input 1 Input 2 Output
True True True
False True False
True False False
False False False

OR Gate (p∨q)

Input 1 Input 2 Output
True True True
False True True
True False True
False False False
These gates are called universal logic gates. This is because thes are the gates that every gate is buit of. Here are some compound gates. The NAND gate is a AND gate and a NOT gate, and the NOR gate is a OR gate with a NOT gate.

NAND Gate (¬[p∧q])

Input 1 Input 2 Output

Transistors

Now, all this logic stuff is very theoretical. How do we use it in the real world? The answer is Transistors. Actually, this is half true. Before transistors, people had these things called vacuum tubes. They were the size of a soda can, and were what was used in the room-sized computers. Back to transistors though, they can be anywhere from 0.5cm tall(minus wires) to 45 nanometers(0.0000025) big, much smaller than a soda can. The way you use transistors for this purpose[1], is like a switch.[See Left] When you apply no current to the middle lead, you can not pass electicity through the other two leads. However, when you do apply electricity to the middle lead, you can pass electricity though the right and left leads. Looking at the illustation on the left, we can see a highly stylised version of a transistor. When you click it, you will see the basic idea.
